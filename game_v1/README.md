# Game

My exploration into game development.

## Building and Running
```sh
sh game_x11_linux.c # compile platform-dependent (X11/Linux) binary
sh game.c # compile shared object (dylib/dll coming soon)
./game
```

## Live Reloading
```sh
./game # while the game is running, make changes to game.c
sh game.c # recompile
```
